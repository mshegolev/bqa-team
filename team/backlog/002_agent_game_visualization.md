# Business Task: Agent Team Visualization / Warcraft-style Map MVP

Create a lightweight static dashboard that represents BQA agents as units on a board/map and shows task flow from business idea to architecture review, development, QA, bug fixing, business acceptance, and done.

Goal:
- make the AI team workflow visible;
- show issue status transitions;
- make demos more engaging;
- avoid heavy game engine complexity.

Constraints:
- static HTML/CSS/JS MVP;
- synthetic data only;
- no backend required.
