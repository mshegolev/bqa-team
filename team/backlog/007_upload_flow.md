# Business Task: Static site upload flow

Implement a browser-only HTML and JavaScript upload flow.

User story:
- user opens the demo page;
- user selects a synthetic marked archive;
- browser reads the archive locally;
- browser displays generated agents, workflows, specs, and recommendations;
- user downloads a generated result archive.

Constraints:
- static site only;
- browser-only processing;
- synthetic fixtures only;
- no backend service.
