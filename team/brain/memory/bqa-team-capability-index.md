# BQA Team Capability Index

## Agents

- Architect turns backlog into bounded GitHub issue specs.
- Go CLI implementer works on focused `bqa-os` implementation tasks.
- QA test engineer validates PRs and creates deduplicated failure bugs.
- Safety drift guard stops broad or unsafe work.
- Team evolution manager improves shared BQA Team assets from sanitized feedback.

## Skills

- QA Memory Pilot creates reusable QA memory packs.
- Autopilot Operations runs and heals long-running team automation.
- Testing-Only Runner applies only the QA persona to a target repo.

## Workflows

- GitHub Team Pipeline handles issue to PR to QA to business acceptance.
- ETL QA Memory Pack handles generated ETL QA pack validation and self-heal.
- BQA Brain Sync exports unified reusable knowledge into git-backed memory.

## Guardrails

- Local-first execution.
- Explicit consent before reading runtime artifacts for self-improvement.
- Validation before success claims.
- Sanitization before brain sync.
